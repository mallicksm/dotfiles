# dotfiles
Contains bash, vim and nvim configurations
initrc/   - bash config
install/  - non-root install scripts
nvim/     - Neovim config
utils/    - common utilities for bash
vim/      - vim config

## Install
1. git clone this repo
2. bash ./dotfiles/dotfiles.sh

## Pre-requisites
1. git clone https://github.com/mallicksm/home
